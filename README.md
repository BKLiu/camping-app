# **露營小助手 (Camping Assistant) 🏕️**

一個專為露友設計的輕量級 Web App，整合了**營地行程管理**、**多人即時分帳**與**地圖導航連動**功能。無需下載，透過瀏覽器即可與夥伴即時協作。

## **✨ 主要特色**

* **📊 全能總覽儀表板**：即時統計總支出、人均費用、營位費與手續費，並視覺化顯示各露友的支出貢獻。
* **🌐 多人即時同步**：整合 Firebase Firestore，一鍵「同步資料」即可將本地紀錄完整推送到雲端，產生專屬 ID 分享給夥伴即時編輯。
* **💰 智能分帳系統**：自動計算「誰該給誰多少錢」的最短還款路徑，並預設依成員**姓名筆劃**自動排序，查找超方便。
* **⛺ 行程與財務細節**：支援紀錄露營日期、營位費/夜衝費（以每帳計費）、手續費及詳細備註，並可一鍵跳轉 Google Maps 導航。
* **📱 響應式與操作優化**：針對手機操作深度優化，支援鍵盤 Enter 快速輸入，內建日期選擇器與防呆機制。

## **🛠️ 版本更新摘要**

### **v1.0.8 (最新版本)**

* **介面簡化**：移除介面上的「(筆畫排序)」提示文字，讓畫面更清爽，但仍維持筆劃排序邏輯。
* **同步優化**：修正並強化「同步資料」邏輯，確保從單機切換至雲端時，已輸入的成員、行程與支出資料會完整上傳，不會遺失。
* **計費調整**：營位費與夜衝費全面改為「每帳」計費制，更符合露營實務。
* **功能新增**：加入「手續費」獨立欄位與「行程日期」選取功能。

## **🚀 快速發佈指南**

### **1. 取得 Firebase 配置**

1. 前往 [Firebase Console](https://console.firebase.google.com/) 建立專案。
2. 啟用 **Firestore Database**（測試模式）與 **Authentication**（匿名登入）。
3. 複製 firebaseConfig 內容。

### **2. 更新與上傳**

1. 開啟 index.html，將Firestore設定貼入指定區塊（約第 260 行）。
2. 上傳至 GitHub Repository 並在 Settings 中開啟 **GitHub Pages** 即可使用。

## **📖 使用小撇步**

* **資料不遺失**：您可以先在單機模式輸入完所有資料，等到要分帳給夥伴時，點擊右上角連線並按「同步資料」，資料會自動帶入雲端。
* **快速儲存**：在大部分輸入框輸入完畢後，直接按手機鍵盤的 **Enter** 鍵即可執行新增或儲存，不需手動點按鈕。
* **姓名排序**：成員名單、分帳建議及支出總覽現在皆預設依**繁體中文筆劃由少到多**排序。
